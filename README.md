# Wardrobe AI 👗

Проект для управления гардеробом с использованием искусственного интеллекта. Позволяет загружать одежду, удалять фон, определять категории и цвета, а также создавать образы.

## 📋 Требования

Для успешного запуска проекта вам понадобятся:
- **Docker Desktop** (для запуска баз данных PostgreSQL и MongoDB)
- **Node.js** версии 18 или выше (для фронтенда)
- **Python** версии 3.10 или выше (для бекенда)

---

## 🚀 Быстрый старт (Windows)

Для удобства в проекте есть скрипт автоматизации, который запускает все необходимые сервисы одной командой.

1. Откройте PowerShell в папке проекта.
2. Запустите скрипт:
   ```powershell
   .\dev.ps1
   ```

**Что делает этот скрипт:**
- Поднимает контейнеры с базами данных (PostgreSQL, MongoDB) в Docker.
- Запускает Backend сервер (FastAPI) в новом окне.
- Запускает ML Worker (обработчик AI задач) в новом окне.
- Запускает Frontend (Vite) в новом окне.

*Если скрипт пишет ошибку безопасности, разрешите выполнение скриптов в текущей сессии:*
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

## 🛠 Ручная установка и запуск

Если вы предпочитаете запускать компоненты вручную, следуйте этой инструкции.

### 1. Запуск баз данных
Проект использует **PostgreSQL** (хранение данных) и **MongoDB** (логи изменений). Запустите их через Docker Compose:

```bash
docker compose up -d db mongo
```

### 2. Настройка Backend (API)

1. Перейдите в папку бекенда:
   ```bash
   cd backend
   ```
2. Создайте виртуальное окружение python:
   ```bash
   python -m venv venv
   ```
3. Активируйте виртуальное окружение:
   - **Windows**: `.\venv\Scripts\activate`
   - **Linux/Mac**: `source venv/bin/activate`
4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
5. Запустите сервер:
   ```bash
   uvicorn main:app --host 0.0.0.0 --port 8000 --reload
   ```
   Сервер запустится на `http://localhost:8000`.

### 3. Запуск ML Worker
Этот компонент отвечает за удаление фона и распознавание одежды. Он должен работать параллельно с бекендом.

1. Откройте **новый терминал**.
2. Перейдите в папку `backend` и убедитесь, что **venv активирован**.
3. Запустите воркер:
   ```bash
   python -m app.ml.worker
   ```

### 4. Настройка Frontend

1. Перейдите в папку фронтенда:
   ```bash
   cd frontend
   ```
2. Установите зависимости (если запускаете впервые):
   ```bash
   npm install
   ```
3. Запустите проект:
   ```bash
   npm run dev
   ```
   Приложение откроется на `http://localhost:3000` (или `http://localhost:5173`, ссылка будет в консоли).

## 🐳 Запуск Backend в Docker

Вы также можете запустить весь бекенд-стек (БД + API + ML Worker) полностью в Docker, без установки Python локально:

```bash
docker compose up --build
```
Поле этого останется запустить только фронтенд (см. пункт 4 выше).

## 🔑 Доступы и порты

- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:8000
- **PostgreSQL**: порт `4242` (user: `postgres`, pass: `12345`)
- **MongoDB**: порт `27017`

## 📁 Структура проекта
- `/backend` — API на FastAPI и ML модули.
- `/frontend` — Интерфейс на React + Vite + Tailwind.
- `/ops` — Конфигурации для деплоя (Nginx и др.).
- `docker-compose.yml` — Описание сервисов для Docker.
