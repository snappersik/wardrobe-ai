# Умный Гардероб (Smart Wardrobe) Landing Page

## О проекте
Минималистичный лендинг для мобильного приложения "Умный Гардероб", которое использует AI для создания образов одежды.

## Стек технологий
- React 18
- TailwindCSS
- Lucide Icons (Static)

## Дизайн
- Основной цвет: #ff91af (Розовый)
- Стиль: Минимализм, чистые линии, много воздуха.
- Шрифт: Inter

## Структура файлов
- `index.html` - Основная точка входа, конфигурация Tailwind.
- `app.js` - Сборка страницы.
- `components/` - Папка с React компонентами (Hero, Features, etc.).

## Правила обновления
При обновлении проекта проверяйте актуальность этого файла. Если добавляются новые секции или меняется дизайн-система, отразите это здесь.
